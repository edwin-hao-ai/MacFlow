Features

Everything your Mac needs. Nothing it doesn't.

No AI. No subscription traps. Open, clean, done.

Safe by design

Cleans what's safe. Leaves everything else alone.

The worst thing a cleaner can do is delete something you needed. MacFlow is built around that fear.

Won't touch active tools

If you're running npm install, building in Xcode, or compiling Rust code right now — MacFlow won't touch those caches. It checks first, every time.

Strict path limits

MacFlow can only delete from a fixed list of known-safe locations. Your home folder, Documents, Desktop, and Downloads are permanently off-limits. No exceptions.

Full history log

Every cleanup is logged locally with timestamps, file sizes, and results. You can always see exactly what was cleaned and when — nothing happens silently.
